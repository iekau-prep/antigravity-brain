# completion_definition.md
Updated: 2026-06-09
Status: Draft

=============================

■ 概要

=============================

本モジュールは、

external_property_search の
完成候補判定条件を定義する。

目的：

何を作るかではなく、

何が成立したら、
external_property_search を
完成候補として閉じられるか固定する。

注意：

完成候補
≠
公開判定

完成候補
≠
実装完了

公開判断はPM責務。

=============================

■ 判定思想

=============================

完成とは、

画面数ではない。

機能数ではない。

---

成立条件：

責務成立

↓

停止しない

↓

判断へ戻れる

---

改善余地は許容する。

=============================

■ EPS-01

外部探索入口成立

=============================

目的：

外部探索から開始できる。

成立条件：

・external到達可能

・探索開始可能

・探索で終了しない

完了：

次導線へ進行可能

=============================

■ EPS-02

入力成立

=============================

目的：

探索結果を受け取れる。

成立条件：

以下いずれか成立。

・URL

・画像

・PDF

・手入力

不足：

許容。

完了：

確認画面へ到達可能

=============================

■ EPS-03

確認成立

=============================

目的：

読む対象固定。

成立条件：

・入力確認可能

・修正可能

・不足許容

・読む押下可能

禁止：

自動決定

完了：

読む責務へ移行可能

=============================

■ EPS-04

property_reader接続成立

=============================

目的：

判断入口へ戻す。

成立条件：

・責務移管成立

・読む後停止なし

・property_reader開始可能

禁止：

OS直通

完了：

property_reader責務開始

=============================

■ EPS-05

decision continuity成立

=============================

目的：

探索で終わらない。

成立条件：

external

↓

property_reader

↓

decision

↓

decision_OS

継続可能。

禁止：

探索終了

完了：

次decision接続可能

=============================

■ EPS-06

recommendation未混入

=============================

目的：

判断主体維持。

成立条件：

以下未発生。

・おすすめ

・順位

・評価点

・自動decision

完了：

判断主体維持

=============================

■ EPS-07

STATE非保存成立

=============================

目的：

責務侵食防止。

成立条件：

以下未保持。

・STATE

・decision_result

・比較状態

・property生成

完了：

接続責務維持

=============================

■ EPS-08

再開破綻なし

=============================

目的：

途中離脱許容。

成立条件：

・入力途中保持成立

・読む後破棄成立

・再入力可能

完了：

停止なし

=============================

■ 最終定義

=============================

external_property_search の完成とは、

探索行動を、

停止させず、

判断へ戻せる責務が成立した状態である。

探索は終点ではない。

判断入口である。
