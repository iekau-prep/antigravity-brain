# 🗄️ Resources Directory

This folder is for reference materials and raw data.

## Examples
- PDFs of market reports.
- CSV files of housing data.
- Images or logos.
- Competitor analysis notes.

## Keep it clean
- If it's something you created, it probably belongs in `content/` or `strategy/`.
- If it's something you found or are using as a reference, it belongs here.
