# 🛠️ Tools Directory

This folder is for scripts, calculators, and automation tools.

## How to use
- If you have a python script, put it here.
- If you have a spreadsheet or a simple HTML calculator, put it here.

## Organization
- It's best to create a subfolder for each tool (e.g., `tools/mortgage-calculator/`).
- Include a `README.md` inside each tool's folder explaining how to run it.
