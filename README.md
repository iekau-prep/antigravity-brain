# Home-Buying Education Platform: AI Brain & Workspace

Welcome to your AI-powered workspace for building a home-buying education platform. This repository serves as the central "brain" for all your tools, content, and strategies.

## 📂 Project Structure

This workspace is organized into four main areas to keep things clean and focused:

### 1. 📚 `content/`
**Goal:** Educational materials for your audience.
- **What goes here:** Blog posts, video scripts, email newsletters, guides.
- **Structure:** Organize by topic (e.g., `financing/`, `search/`, `closing/`).

### 2. 🛠️ `tools/`
**Goal:** Utilities to help you work faster and smarter.
- **What goes here:** Calculators (e.g., "Mortgage Affordability"), automation scripts, data analysis tools.
- **Structure:** Each tool should have its own folder with a `README.md` explaining how to use it.

### 3. 🧠 `strategy/`
**Goal:** The "Why" and "How" behind your platform.
- **What goes here:** User personas, market research, content calendars, long-term roadmap.
- **Key File:** `roadmap.md` (Track your high-level goals here).

### 4. 🗄️ `resources/`
**Goal:** Reference materials and raw data.
- **What goes here:** Industry reports, competitor analysis, design assets, raw data files.

## 🚀 Getting Started

1.  **Plan**: Check `strategy/roadmap.md` to see what next.
2.  **Create**: Start writing in `content/` or building in `tools/`.
3.  **Organize**: Keep this root folder clean. If a file doesn't fit in one of the 4 main folders, ask yourself if it belongs in a new category or if it can be archived.
