# 📚 Content Directory

This folder is for all your educational materials.

## Sub-folders
- Create folders for topics like `financing/`, `search/`, etc.
- Or organize by type: `scripts/`, `posts/`, `guides/`.

## Workflow
1.  Use `task.md` or `strategy/roadmap.md` to identify what content to create next.
2.  Draft it here in markdown files (e.g., `financing/first-time-guide.md`).
3.  Use this as the source for your blog, videos, or newsletter.
